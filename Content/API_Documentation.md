## API Documentation

## Search Repositories
- **URL**: `https://api.github.com/search/repositories`.
- **Parameters**:
  - `q`: search query (mandatory).
  - `sort`: sort (e.g., `stars`).
  - `order`: order (`asc` or `desc`).

## Commits
- **URL**: `https://api.github.com/repos/{owner}/{repo}/commits`.
- **Parameters**:
  - `per_page`: number of entries per page.
  - `page`: page number.
